# ポモドーロタイマー 🍅

プログラミングとGit/GitHubを学ぶために作った、シンプルなポモドーロタイマーです。

## ポモドーロテクニックとは

「**25分の作業 → 5分の休憩**」を1セットとして繰り返す時間管理術です。
集中と休憩のリズムを作ることで、勉強や仕事がはかどります。

## 使い方

1. `index.html` をブラウザで開く
2. **スタート**でカウントダウン開始
3. **ストップ**で一時停止、**リセット**で最初(25:00)に戻す

## 使っている技術

- HTML … ページの構造
- CSS … 見た目
- JavaScript … タイマーの動き

## 学習メモ

このプロジェクトは学習用です。作りながら次のことを学びます。

- プログラミングの基礎（変数・関数・繰り返し・条件分岐・イベント処理）
- Gitの基礎（add / commit で変更を記録する）
- GitHubの基礎（push でネットに保存、GitHub Pagesで公開する）
